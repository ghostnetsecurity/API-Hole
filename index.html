Impact Assessment

Missing nonce validation in the plugin's parseData function allows unauthenticated attackers to craft requests that, when executed by an administrator, trigger exports of sensitive data including user information, email addresses, and WooCommerce data.

Data Exposure Vector: Successful exploitation results in the generation of CSV files containing sensitive information, stored in publicly accessible directories, creating a direct data breach vector without requiring authentication.
