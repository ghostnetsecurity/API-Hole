Vulnerability Analysis

This platform provides diagnostic analysis of CVE-2025-13606, a Cross-Site Request Forgery (CSRF) vulnerability affecting the WordPress "Export All Posts, Products, Orders, Refunds & Users" plugin.

Impact Assessment

Missing nonce validation in the plugin's parseData function allows unauthenticated attackers to craft requests that, when executed by an administrator, trigger exports of sensitive data including user information, email addresses, and WooCommerce data.

Data Exposure: Successful exploitation results in the generation of CSV files containing sensitive information, stored in publicly accessible directories, creating a direct data breach vector.
